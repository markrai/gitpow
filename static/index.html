Per-Branch Seek Depth
Controls how many commits are fetched per branch when using the All-branches Vertical/Horizontal map views. Default is 100.

Upper bound on total commits loaded for graph rendering in All-branches Vertical/Horizontal map views. Default is 2000.

Maximum number of branches to include when viewing all branches in Vertical/Horizontal map views. Older branches beyond this limit are omitted. Default is 50.
